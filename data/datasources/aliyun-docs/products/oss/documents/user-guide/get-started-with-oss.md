对象存储OSS是阿里云提供的海量、安全、低成本的云存储服务，用于存放和管理各种数据文件，并提供多种访问方式。

## 选择合适的工具

OSS提供多种工具满足不同场景需求，请根据实际场景选择适合的工具。

| 工具名称 | 适用场景 | 入门指引 |
| --- | --- | --- |
| OSS 控制台 | 无需安装软件，通过浏览器登录即可管理 OSS 资源。适合快速操作、资源监控、权限配置、文件上传下载等日常管理任务。 | [控制台快速入门](products/oss/documents/user-guide/console-quick-start.md) |
| ossbrowser | 在 Windows、macOS 或 Linux 桌面上管理 OSS 资源。提供图形化界面，支持拖拽上传、断点续传，适合日常文件管理。 | [图形化管理工具](products/oss/documents/user-guide/ossbrowser.md) [ossbrowser 2.0](products/oss/documents/user-guide/ossbrowser.md) [快速入门](products/oss/documents/user-guide/ossbrowser.md) |
| ossutil | 通过命令行管理 OSS 资源，支持批量操作、自动化脚本和定时任务。适合需要高效处理大量文件或集成到自动化流程的场景。 | [命令行工具](products/oss/documents/command-line-tools-ossutil-quickstart.md) [ossutil](products/oss/documents/command-line-tools-ossutil-quickstart.md) [快速入门](products/oss/documents/command-line-tools-ossutil-quickstart.md) |
| SDK | 在应用程序中集成 OSS 功能，提供 Java、Python、PHP、Go 等 12 种编程语言的 SDK 包。适合需要在应用中读写 OSS 数据的开发场景。 | [SDK](products/oss/documents/user-guide/oss-sdk-quick-start.md) [快速入门](products/oss/documents/user-guide/oss-sdk-quick-start.md) |
| ossfs | 将 OSS Bucket 挂载到 Linux 系统，像操作本地文件系统一样使用 OSS。适合需要通过标准文件接口访问 OSS，无需修改现有应用代码的场景。 | [ossfs](products/oss/documents/user-guide/ossfs-quick-start.md) [快速入门](products/oss/documents/user-guide/ossfs-quick-start.md) |
| OSS connector for AI/ML | 在 PyTorch 等 AI 训练框架中直接访问 OSS 数据集，无需下载到本地，支持训练进度自动保存和恢复。适合机器学习模型训练场景。 | [OSS Connector for AI/ML](products/oss/documents/user-guide/oss-connector-for-ai-ml-quick-start.md) [快速入门](products/oss/documents/user-guide/oss-connector-for-ai-ml-quick-start.md) |
| 在 Cursor 中使用 OSS 文档 | 在 Cursor 编辑器中快速查阅 OSS 文档，边开发边参考。适合使用 Cursor 进行开发，需要随时查阅 OSS API 和功能文档的场景。 | [在](products/oss/documents/user-guide/use-the-oss-document-in-cursor.md) [Cursor](products/oss/documents/user-guide/use-the-oss-document-in-cursor.md) [中使用](products/oss/documents/user-guide/use-the-oss-document-in-cursor.md) [OSS](products/oss/documents/user-guide/use-the-oss-document-in-cursor.md) [文档](products/oss/documents/user-guide/use-the-oss-document-in-cursor.md) |

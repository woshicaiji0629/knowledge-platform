# 在阿里云App上使用CDN

更新时间：

复制 MD 格式

[产品详情](https://www.aliyun.com/product/cdn)

[我的收藏](https://help.aliyun.com/my_favorites.html)

在iOS、Android等移动设备中，您可以通过阿里云App登录CDN控制台进行配置。本文为您介绍在移动端进行数据监控、预付费资源包购买、域名启用/停用、刷新预热、IP归属查询等常用热门场景的操作方式。

## 前提条件

- 

已经在移动端安装了阿里云App，并已注册登录，具体请参见[在阿里云](https://help.aliyun.com/zh/document_detail/185103.html#topic-2042416)[App](https://help.aliyun.com/zh/document_detail/185103.html#topic-2042416)[上登录账号](https://help.aliyun.com/zh/document_detail/185103.html#topic-2042416)。

- 

您已添加CDN加速域名，具体请参见[添加加速域名](products/cdn/documents/add-a-domain-name.md)。

## 登录入口

- 

打开阿里云App，选择控制台页签。

- 

在上方区域选择更多控制台，进入到全部控制台页面。

- 

在全部控制台页面，选择网络与CDN页签，产品列表中选择CDN。

说明

如果网络与CDN页签中没有CDN产品，您可以在我的控制台中查看，或者直接在首页搜索CDN。

## 数据监控

阿里云App支持随时随地数据查询。您可以随时随地通过移动端查看CDN业务运行情况。

选择概览，在下方区域您可以查看请求数、带宽统计、流量统计等指标。

## 资源包购买

当资源包快要耗尽时，如果身边没有电脑无法购买资源包，此时您可以通过手机支付购买，方便快捷。

选择资源包，在下方区域单击购买资源包，按照界面操作指引完成购买操作即可。收费规则和Web端一致，具体请参见[资源包介绍](products/cdn/documents/product-overview/resource-plan-overview.md)。

## 域名启用/停用

您的域名可能会遭受攻击，或者出现不在预期之内的流量突增，这带来短时间内的消费增高。当出现这些情况时，您可以评估自身情况后，通过阿里云App暂时停止加速服务，减少不必要的损失。

选择域名管理，在下方对应域名区域右上角单击按钮，您可以启动或停止该域名。

- 

停用域名

- 

启用域名

## 刷新预热

如果您的源站存在不合规内容，当您删除源站资源后，由于CDN节点存在缓存，资源仍可以被访问到，可通过URL刷新功能更新缓存资源。

选择刷新缓存/预热缓存，您可以在界面中对加速资源进行预热或刷新操作，具体功能和Web端相同，请参见[刷新和预热资源](products/cdn/documents/user-guide/refresh-and-prefetch-resources.md)。

## IP归属查询

用户排查故障时，可以通过IP归属来判断故障的所属区域，无论是在办公场景，还是在下班场景，IP查询工具可以帮助您快速的定位问题。

选择IP查询工具，您可以在界面中检测指定的IP地址是否为阿里云CDN节点的IP，以及IP所属地域和运营商。具体功能和Web端相同，请参见[检测](products/cdn/documents/user-guide/does-the-ip-belong-to-cdn-pops.md)[IP](products/cdn/documents/user-guide/does-the-ip-belong-to-cdn-pops.md)[地址](products/cdn/documents/user-guide/does-the-ip-belong-to-cdn-pops.md)。

阿里云CDN 2014旧版API接口已经于2021年2月25日零点起逐步下线。如果您的系统仍在调用2014旧版API，请尽快更新至2018版API，避免因旧版API下线而导致系统服务不可用。如果您有任何疑问，可以提交工单。

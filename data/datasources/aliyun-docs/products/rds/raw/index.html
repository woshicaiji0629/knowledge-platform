云数据库 RDS产品简介

    产品系列

    RDS MySQL数据库

    RDS MySQL是基于阿里巴巴MySQL源码分支的在线数据库服务，经过双十一高并发、大数据量的考验，拥有优良的性能。

    RDS PostgreSQL数据库

    RDS PostgreSQL的优点主要集中在对SQL规范的完整实现以及丰富多样的数据类型支持，包括JSON数据、IP数据和几何数据等。完美支持事务、子查询、多版本控制（MVCC）、数据完整性检查等特性。

    RDS SQL Server数据库

    RDS SQL Server拥有微软正版的License授权，同时还拥有高可用架构和任意时间点的数据恢复功能，强力支撑各种企业应用。

    RDS MariaDB数据库

    RDS MariaDB基于MariaDB提供企业级性能。

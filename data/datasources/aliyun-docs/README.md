# Aliyun Docs Crawled Data

This directory stores Git-tracked crawler output for the Aliyun Docs data source.

```text
products/
  <product>/
    raw/        Raw HTML files.
    documents/  Local Markdown files.
    metadata/   Per-page metadata JSON files.
seeds.json      First-batch product seed URLs.
manifest.json   Crawl manifest generated by the crawler.
cleaned/        Cleaned Markdown files accepted for product Q&A.
chunks/         Markdown chunks generated from cleaned documents.
chunk-manifest.json
                Chunk manifest for embedding and vector indexing.
```

Do not put Docker service volumes here. Runtime data for MySQL, Redis, vector stores, and similar services must use a separate runtime directory.

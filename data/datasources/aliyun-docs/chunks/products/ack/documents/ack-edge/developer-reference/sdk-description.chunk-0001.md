## SDK版本说明
容器服务 Kubernetes 版[OpenAPI](https://next.api.aliyun.com/api/CS/2015-12-15/OpenAckService)[门户](https://next.api.aliyun.com/api/CS/2015-12-15/OpenAckService)提供V1.0 SDK和V2.0 SDK。V2.0 SDK不涉及接口功能变更，主要是为了提升SDK的健壮性及使用体验。V1.0 SDK已不再维护，推荐您使用V2.0 SDK。
说明
V1.0 SDK即原版SDK，V2.0 SDK即升级版SDK，版本对比详情请参见[V2.0 SDK 和 V1.0 SDK](https://help.aliyun.com/zh/sdk/product-overview/differences-between-v1-and-v2-sdks)。

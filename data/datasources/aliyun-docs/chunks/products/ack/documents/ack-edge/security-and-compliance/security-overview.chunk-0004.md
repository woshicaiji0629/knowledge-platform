## 节点漏洞
节点漏洞检查默认开启。
在安全概览页面下方，单击节点漏洞页签，查看节点漏洞列表，包含对应的节点池以及影响该节点池中的节点数，然后单击修复即可跳转至节点池详情页面进行漏洞修复。关于节点池CVE漏洞修复，请参见[修复节点池操作系统](../../ack-managed-and-ack-dedicated/user-guide/cve-patching.md)[CVE](../../ack-managed-and-ack-dedicated/user-guide/cve-patching.md)[漏洞](../../ack-managed-and-ack-dedicated/user-guide/cve-patching.md)。
说明
漏洞修复完成后，安全概览页面的相关数据还需等待24小时才会刷新。

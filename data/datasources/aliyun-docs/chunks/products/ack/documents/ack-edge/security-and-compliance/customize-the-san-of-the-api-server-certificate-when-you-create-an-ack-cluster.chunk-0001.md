## 前提条件
已创建ACK托管集群、ACK专有集群或ACK Serverless集群。具体操作，请参见[创建](../../ack-managed-and-ack-dedicated/user-guide/create-an-ack-managed-cluster-2.md)[ACK](../../ack-managed-and-ack-dedicated/user-guide/create-an-ack-managed-cluster-2.md)[托管集群](../../ack-managed-and-ack-dedicated/user-guide/create-an-ack-managed-cluster-2.md)、[创建](../../ack-managed-and-ack-dedicated/user-guide/create-an-ack-dedicated-cluster.md)[ACK](../../ack-managed-and-ack-dedicated/user-guide/create-an-ack-dedicated-cluster.md)[专有集群（已停止新建）](../../ack-managed-and-ack-dedicated/user-guide/create-an-ack-dedicated-cluster.md)或[创建集群](../../serverless-kubernetes/user-guide/create-an-ask-cluster-2.md)。
重要
ACK Serverless集群不支持在新建集群时自定义SAN字段，仅支持在已有集群中更新SAN字段。
ACK专有集群仅支持在新建集群时自定义SAN字段，不支持在已有集群中更新SAN字段。

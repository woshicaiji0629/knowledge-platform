## 支持的操作系统
关于容器服务 Kubernetes 版支持的操作系统，请参见[操作系统](../ack-managed-and-ack-dedicated/user-guide/overview-of-os-images.md)。
重要
如果操作系统版本完全停止维护（EOL），阿里云会同时停止对使用相应操作系统版本的ECS实例提供技术支持服务。Alibaba Cloud Linux 2、CentOS分别于2024年03月31日00:00:00、2024年06月30日00:00:00 EOL。为了获得快速高效的技术支持和保障业务稳定，建议您使用操作系统Alibaba Cloud Linux 3或ContainerOS。关于变更内容及解决方案，请参见[【产品变更】停止维护](announcement-on-stopping-maintenance-of-alibaba-cloud-linux-2-and-centos-7.md)[Alibaba Cloud Linux 2、CentOS 7](announcement-on-stopping-maintenance-of-alibaba-cloud-linux-2-and-centos-7.md)[的公告](announcement-on-stopping-maintenance-of-alibaba-cloud-linux-2-and-centos-7.md)。

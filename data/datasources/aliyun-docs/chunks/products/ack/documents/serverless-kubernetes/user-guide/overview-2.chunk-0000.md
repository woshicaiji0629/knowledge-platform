# Nginx Ingress概述

重要
由于Ingress NGINX开源项目于2026年3月后停止维护更新，容器服务 Kubernetes 版将停止Nginx Ingress Controller组件维护，请充分了解使用风险。更多详细内容，请参见[【产品公告】关于停止维护](../../product-overview/product-announcement-announcement-on-end-of-maintenance-for-nginx-ingress-controller.md)[Nginx Ingress Controller](../../product-overview/product-announcement-announcement-on-end-of-maintenance-for-nginx-ingress-controller.md)[组件的公告](../../product-overview/product-announcement-announcement-on-end-of-maintenance-for-nginx-ingress-controller.md)。

## 配置节点池OS参数
您可以在节点池维度配置sysctl参数和Transparent Hugepage（THP）参数。这些参数均可通过修改配置文件的方式配置；[THP](custom-node-pool-os-parameters.md)[参数](custom-node-pool-os-parameters.md)和[部分](custom-node-pool-os-parameters.md)[sysctl](custom-node-pool-os-parameters.md)[参数](custom-node-pool-os-parameters.md)还支持通过控制台或OpenAPI配置。

重要
仅阿里云账号（主账号）或[RAM](../../../../ram/documents/create-admin-user.md)[管理员账号](../../../../ram/documents/create-admin-user.md)可以完成自动授权，普通RAM用户没有授权操作的权限。如果系统提示权限不足，请将账号切换到阿里云（主账号）或RAM管理员账号完成授权。

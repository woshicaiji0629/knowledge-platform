## 删除Tag
在高级参数配置中将该Tag设置为空字符串，从而实现删除Tag目的。
以下示例中AGENT_TAG的value设置为空字符串，该Tag不会显示。具体操作，请参见[创建/修改 Tag](manage-loongcollector-to-collect-tags.md)[名称](manage-loongcollector-to-collect-tags.md)。
{ "PipelineMetaTagKey": { "HOST_NAME": "sourceName", "AGENT_TAG": "", "HOST_ID": "__default__", "CLOUD_PROVIDER": "__default__" } }

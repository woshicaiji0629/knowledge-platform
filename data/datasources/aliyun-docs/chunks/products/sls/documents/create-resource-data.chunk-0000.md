## 操作步骤
登录[日志服务控制台](https://sls.console.aliyun.com)。
进入资源数据管理页面。
在Project列表区域，单击任意一个Project。
在左侧导航栏中，单击告警。
在告警中心页面，选择其它>资源数据。
单击创建，配置资源数据的信息。

| 参数 | 描述 |
| --- | --- |
| ID | 资源数据唯一标识，不可重复。 |
| 名称 | 资源数据的名称。 |
| 字段信息 | 资源数据的字段信息，您可以单击 增加多个字段、描述信息和字段类型。字段类型支持字符串、浮点数和整数。例如 ip_black_list、IP 地址黑名单、字符串。 |

单击确认。

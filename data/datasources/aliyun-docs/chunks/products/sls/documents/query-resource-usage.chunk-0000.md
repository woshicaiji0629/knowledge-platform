## 操作步骤
登录[费用与成本](https://billing-cost.console.aliyun.com)。
在左侧导航栏，选择账单>账单详情。
在用量明细页签中，完成如下设置。

| 参数 | 说明 |
| --- | --- |
| 产品名称 | 选择 sls 。 |
| 计量规格 | 选择 日志服务 。 |
| 使用时间 | 选择您要查询的时间范围。 仅支持自然月内的用量查询。例如开始时间为 2022-02-01，结束时间最大为 2022-02-28，不支持选择 2022-03-01。 |
| 计量粒度 | 选择 小时 。 |

单击导出。
待文件导出成功后，您可以将文件下载到本地，查看日志服务的使用明细。

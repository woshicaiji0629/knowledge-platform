## 请求参数
该命令的必选和特有参数描述如下。

| 参数名称 | 数值类型 | 是否必选 | 示例值 | 描述 |
| --- | --- | --- | --- | --- |
| --project_name | String | 是 | aliyun-test-project | Project 名称。 |
| --logstore_name | String | 是 | logstroe-a | Logstore 名称。 |
| --index_detail | JSON Object | 是 | file://./indexdetail.json | 索引配置文件路径。更多信息，请参见 [创建索引](../create-indexes.md) 和 [GitHub](https://github.com/aliyun/aliyun-log-cli/blob/master/tests/create_index.json) 。 |

关于该命令的全局参数，请参见[全局参数](global-parameters.md)。

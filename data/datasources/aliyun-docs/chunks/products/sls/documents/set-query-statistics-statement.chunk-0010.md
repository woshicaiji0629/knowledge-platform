### 左联
查询统计0结果

| a | b |
| --- | --- |
| a1 | b1 |
| a2 | b2 |
| a3 | b3 |

查询统计1结果

| a | b | c |
| --- | --- | --- |
| a1 | b11 | c1 |
| a2 | b22 | c2 |

集合操作结果
当选择集合操作为左联，$0.a == $1.a时，集合操作结果如下：

| a | $0.b | $1.b | c |
| --- | --- | --- | --- |
| a1 | b1 | b11 | c1 |
| a2 | b2 | b22 | c2 |
| a3 | b3 | 无 | 无 |

# 通过购买页自定义购买ECS
与快速购买实例相比，自定义购买可根据业务场景灵活地选择配置，如镜像类型、实例规格、存储、带宽、安全组等。本文介绍如何自定义购买实例。
重要
如果您刚接触ECS不久，想要快速上手购买并使用ECS实例，我们为您准备了一个较简单的示例，请参见[控制台自定义购买并使用](use-the-ecs-instance-in-the-console.md)[ECS](use-the-ecs-instance-in-the-console.md)[实例](use-the-ecs-instance-in-the-console.md)，它将帮助您更直观地理解整个流程。在初步了解ECS后，您可以通过阅读本文，了解自定义购买实例时更加详细的配置说明。

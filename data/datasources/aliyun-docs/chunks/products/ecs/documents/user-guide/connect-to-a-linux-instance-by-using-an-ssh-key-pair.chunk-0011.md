## 常见问题
- 如何配置安全组规则以放行22端口？
在实例所在安全组[添加](start-using-security-groups.md)如下安全组规则：

| 授权策略 | 协议 | 访问来源 | 访问目的(本实例) |
| --- | --- | --- | --- |
| 允许 | 自定义 TCP | 输入本地客户端的公网 IP 地址。 重要 若使用 0.0.0.0/0 ，表示允许任意 IP 访问远程服务端口，存在安全风险，请谨慎使用。 | SSH(22) 如果修改了实例的 SSH 服务的端口，需调整为实际端口。 |

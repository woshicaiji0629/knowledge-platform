## API
可以通过调用以下API重启一台或多台ECS实例。
重启一台处于运行中（Running）状态的ECS实例：[重启实例](../developer-reference/api-ecs-2014-05-26-rebootinstance.md)。
重启一台或多台处于运行中（Running）状态的ECS实例：[批量重启实例](../developer-reference/api-ecs-2014-05-26-rebootinstances.md)。
重新启动已过期或欠费回收的ECS实例：[重新激活已过期或欠费回收的实例](../developer-reference/api-ecs-2014-05-26-reactivateinstances.md)。
若需定时重启实例，可参考[系统运维管理](https://oos.console.aliyun.com/cn-hangzhou/template/public/detail/ACS-ECS-ScheduleToRebootInstances)[OOS-公共任务模板-定时重启任务](https://oos.console.aliyun.com/cn-hangzhou/template/public/detail/ACS-ECS-ScheduleToRebootInstances)进行配置。

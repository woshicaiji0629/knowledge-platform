### 准备工作
实例公网IP地址：在[ECS](https://ecs.console.aliyun.com/server/region)[控制台-实例](https://ecs.console.aliyun.com/server/region)，找到目标实例进入详情，在配置信息区域找到公网IP。
准备实例登录凭证：为实例[设置密码](instance-logon-credential-management.md)。
配置安全组：为实例关联的安全组[配置入方向规则](connect-to-a-linux-instance-by-using-an-ssh-key-pair.md)，允许本地IP通过RDP（3389端口）访问实例。

## 后续步骤
连接实例
支持通过多种方式连接实例，包括Workbench、VNC和第三方客户端工具。更多信息，请参见[选择](connect-to-instance.md)[ECS](connect-to-instance.md)[远程连接方式](connect-to-instance.md)。
初始化数据盘
如果您随实例创建了数据盘，必须先对数据盘进行分区格式化才能正常使用。具体操作，请参见[初始化数据盘（Linux）](initialize-a-data-disk-whose-size-does-not-exceed-2-tib-on-a-linux-instance.md)和[初始化数据盘（Windows）](initialize-a-data-disk-up-to-2-tib-in-size-on-a-windows-instance.md)。
部署环境、搭建网站、搭建应用
创建实例后，您可以使用ECS实例部署环境、搭建网站和应用等。更多信息，请参见[搭建环境](build-a-software-development-environment.md)、[搭建网站](build-a-website.md)、[搭建应用](build-an-application.md)。

实例主网卡的安全组如何更换/添加？
安全组实际作用在ECS实例的[弹性网卡](eni-overview.md)上。ECS实例详情页面上安全组页签中配置的安全组，即为实例主网卡的安全组。可参考[为实例关联安全组](start-using-security-groups.md)更改实例主网卡关联的安全组。
安全组规则设置为全部拒绝后，如何允许阿里云内部服务访问？
添加入方向安全组规则，允许来自100.64.0.0/10地址段的访问。阿里云使用该保留地址段对实例进行健康检查和可用性监控。

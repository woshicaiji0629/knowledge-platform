## 系统盘和数据盘的常用操作对比

| 生命周期 | 系统盘 | 数据盘 |  |
| --- | --- | --- | --- |
| 创建阶段 | 单独创建云盘 | 不支持 说明 系统盘只能随实例创建，创建后自动完成挂载及初始化，无需额外操作。 | 支持 |
| 挂载云盘 | 不涉及 | 需要执行 |  |
| 初始化云盘 | 不涉及 | 需要执行 |  |
| 运维阶段 | 更换操作系统 | 支持 | 不支持 说明 系统盘包含用于启动该实例的镜像，因此数据盘不涉及更换操作系统操作，此外运维阶段操作类似。 |
| 其他操作 | 支持 | 支持 |  |
| 释放阶段 | 单独释放云盘 | 不支持 说明 系统盘只能随实例释放，无法单独释放。系统盘和数据盘均可以关闭随实例释放，区别在于，在实例释放后系统盘会作为数据盘保留下来，失去系统盘属性。 | 支持 |

## 更多功能
除了会话管理功能外，会话管理CLI（ali-instance-cli）还有以下功能：
[访问无公网的实例（端口转发实现）](perform-port-forwarding-by-using-ali-instance-cli.md)
您可以通过ali-instance-cli的端口转发功能，将实例的某个端口映射到您个人计算机的某个端口，支持无公网实例。由此功能，您可以实现免代理，免跳板机的访问无公网实例上的服务。
[注册临时](register-a-public-key-and-connect-to-an-instance-with-the-key-by-using-ali-instance-cli.md)[SSH](register-a-public-key-and-connect-to-an-instance-with-the-key-by-using-ali-instance-cli.md)[公钥](register-a-public-key-and-connect-to-an-instance-with-the-key-by-using-ali-instance-cli.md)
如果您使用SSH连接实例，您可以选择通过该功能向目标实例中注册临时公钥，此时，您可以通过与之对应的私钥连接实例。

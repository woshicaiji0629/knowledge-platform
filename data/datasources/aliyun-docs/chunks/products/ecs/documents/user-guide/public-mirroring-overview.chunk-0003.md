## Alibaba Cloud Linux镜像
Alibaba Cloud Linux 3、Alibaba Cloud Linux 3 Pro和Alibaba Cloud Linux 2是阿里云自主研发的Linux系统镜像，属于公共镜像。Alibaba Cloud Linux支持多种ECS实例规格，包括弹性裸金属服务器。Alibaba Cloud Linux中默认搭载了阿里云CLI及其他软件包，方便您使用。关于阿里云CLI的更多信息，请参见[什么是阿里云](https://help.aliyun.com/zh/cli/what-is-alibaba-cloud-cli#concept-rc3-qrc-bhb)[CLI](https://help.aliyun.com/zh/cli/what-is-alibaba-cloud-cli#concept-rc3-qrc-bhb)。如果您要从其他Linux版本更换为Alibaba Cloud Linux，可以通过创建新实例或更换操作系统（系统盘）的方式进行更换。更多信息，请参见[Alibaba Cloud Linux](alibaba-cloud-linux.md)。

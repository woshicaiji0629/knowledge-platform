# Redis开源版小版本发布日志
为提升用户体验，云数据库 Tair（兼容 Redis）会不定期地发布小版本，用于丰富云产品功能或修复已知缺陷。您可以参阅本文了解Redis开源版小版本的更新说明，选择在业务低峰期升级实例的小版本。
Redis开源版7.0、6.0、5.0或4.0，完全兼容对应的Redis大版本，并向下兼容。
重要
本文仅包含Redis开源版实例的小版本发布日志，关于Tair（企业版）实例的小版本发布日志（包含内存型、持久内存型和磁盘型）请参见[Tair](apsaradb-for-redis-enhanced-edition-1.md)[小版本发布日志](apsaradb-for-redis-enhanced-edition-1.md)。

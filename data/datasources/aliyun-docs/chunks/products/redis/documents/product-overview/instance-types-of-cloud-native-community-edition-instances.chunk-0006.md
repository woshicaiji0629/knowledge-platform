## 最大连接数说明
代理模式：实例最大连接数的计算规则为：min(Proxy节点数*12万, 50万）。因此上限为50万，如业务连接数需求超过此限制，建议采用集群直连模式，连接数可随分片数量线性扩展。

| 分片数 | Proxy 节点数 | 最大连接数 |
| --- | --- | --- |
| 1 | 2 | 240,000 |
| 2 | 2 | 240,000 |
| 3 | 3 | 360,000 |
| 4 | 3 | 360,000 |
| 5~6 | 3 | 360,000 |
| 7~8 | 4 | 480,000 |
| ≥9 | ≥5 | 500,000（上限） |

直连模式：若单分片最大连接数的上限为10,000，实例最大连接数的上限为：分片数 * 10,000。

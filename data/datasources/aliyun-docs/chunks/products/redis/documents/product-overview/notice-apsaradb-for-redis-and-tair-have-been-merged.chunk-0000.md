# 【通知】云数据库Redis版与云原生内存数据库Tair已合并
云数据库Redis版与云原生内存数据库Tair已于2024年09月27日合并为云数据库 Tair（兼容 Redis），未来将统一以Tair品牌继续提供服务与支持。
您可以通过云数据库 Tair（兼容 Redis）控制台、产品详情页、文档、OpenAPI等继续使用Tair（企业版）与Redis开源版产品。感谢您的关注与支持，我们将以新的产品品牌继续为您提供更加优秀的产品和服务！
关于更多此次合并的信息，请参见下方的常见问题解答，如有任何疑问请[提交工单](https://selfservice.console.aliyun.com/ticket/category/redis/today)与我们联系。

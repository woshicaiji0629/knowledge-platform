## 操作步骤
访问[实例列表](https://kvstore.console.aliyun.com/Redis/instance/cn-hangzhou)，在上方选择地域，然后单击目标实例ID。
在实例信息页面的连接信息区域，定位至要修改的连接类型，单击其操作列的修改。
在右侧弹出的面板中，设置新的连接地址和端口号。

| 配置 | 说明 |
| --- | --- |
| 连接地址 | 目前仅支持修改连接地址的前缀（前缀默认为实例 ID）。 自定义前缀需由小写英文字母、数字组成，但必须以小写字母开头，长度为 8~40 个字符。 |
| 端口号 | 可在修改连接地址的同时，修改端口，范围为 1~65535。 |

单击确定。

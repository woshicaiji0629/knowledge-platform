## 实例的网络类型

| 网络类型 | 说明 |
| --- | --- |
| [专有网络](../../../vpc/documents/what-is-vpc.md) （推荐） | 专有网络 VPC（Virtual Private Cloud）是您专有的云上私有网络。不同的专有网络之间二层逻辑隔离，拥有较高的安全性和性能。客户端部署在 [ECS](../../../ecs/documents/user-guide/what-is-ecs.md) [实例](../../../ecs/documents/user-guide/what-is-ecs.md) 上时，通过专有网络连接至 Tair （以及 Redis 开源版 ） 实例，可获得更高的安全性和更低的网络延迟。 |
| 经典网络 | 经典网络中的云服务在网络上不进行隔离，只能依靠云服务自身的安全组或白名单策略来阻挡非法访问。 |

说明
客户端为专有网络，实例为经典网络时，客户端无法连接至实例。将实例切换至与客户端相同的专有网络后，可快速实现相互通信。

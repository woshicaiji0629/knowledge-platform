## 提高CDN的访问安全性
为防止域名因受到攻击等原因产生突发高带宽，您可以设置监控报警，实时监控带宽的变化情况，或者通过带宽封顶功能，设置带宽上限，防止产生过高的带宽。

| 场景 | 说明 | 配置 |
| --- | --- | --- |
| 限制带宽过高 | 通过带宽封顶功能，设置带宽上限，当检测到某个统计周期的带宽超出您设置的带宽上限时，CDN 将停止为该域名提供加速服务，且该域名会被解析到无效地址 offline.***.com ，无法被继续访问。 | [配置带宽封顶](user-guide/configure-bandwidth-caps.md) |
| 通过在云监控产品中创建报警规则，实现对网络带宽的报警监控，帮助您及时了解带宽异常并快速进行处理。 | [设置报警](user-guide/set-an-alert-rule.md) |  |

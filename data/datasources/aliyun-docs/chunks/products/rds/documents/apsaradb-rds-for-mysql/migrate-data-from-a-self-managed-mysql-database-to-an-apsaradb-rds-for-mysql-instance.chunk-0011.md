### 创建数据库账号并授权
使用DTS进行数据迁移时，源库账号和目标库账号必须具备如下权限：

| 数据库账号 | 库表结构迁移 | 全量迁移 | 增量迁移 |
| --- | --- | --- | --- |
| 自建 MySQL 账号 | SELECT 权限 | 待迁移对象的 SELECT 权限。 REPLICATION CLIENT、REPLICATION SLAVE、SHOW VIEW 权限。 建库建表的权限，以允许 DTS 创建库 test ，用于推进 Binlog 位点。 |  |
| RDS MySQL 账号 | 读写权限 |  |  |

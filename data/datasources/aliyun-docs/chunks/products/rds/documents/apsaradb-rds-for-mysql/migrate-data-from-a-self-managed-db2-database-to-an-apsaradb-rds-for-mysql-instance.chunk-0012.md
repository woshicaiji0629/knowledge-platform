说明
在迁移任务正式启动之前，会先进行预检查。只有预检查通过后，才能成功启动迁移任务。
如果预检查失败，单击具体检查项后的，查看失败详情。
您可以根据提示修复后重新进行预检查。
如无需修复告警检测项，您也可以选择确认屏蔽、忽略告警项并重新进行预检查，跳过告警检测项重新进行预检查。
预检查通过后，单击下一步。
在购买配置确认页面，选择链路规格并选中数据传输（按量付费）服务条款。
单击购买并启动，迁移任务正式开始。

等待迁移任务的增量迁移再次进入无延迟后，手动结束迁移任务。

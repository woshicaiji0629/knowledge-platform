# 文本生成模型API参考
百炼提供多种接口用于调用文本生成模型：
[OpenAI 兼容 Chat Completions](qwen-api-via-openai-chat-completions.md)：与 OpenAI 客户端库直接兼容，迁移现有应用或接入第三方工具的成本最低。
[OpenAI](openai-compatible-responses.md)[兼容-Responses](openai-compatible-responses.md)：内置联网搜索、代码解释器和网页内容提取工具，自动管理对话历史，无需手动维护。
[Anthropic](anthropic-api-messages.md)[兼容-Messages](anthropic-api-messages.md)：兼容 Anthropic Messages API，支持思考和工具调用。
[DashScope](qwen-api-via-dashscope.md)：百炼原生接口，提供最完整的功能集和参数支持。

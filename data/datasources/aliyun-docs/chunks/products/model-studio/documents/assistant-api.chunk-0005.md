### 模型支持
Assistant API 支持千问的多款主流模型。您可以前往[模型广场](https://bailian.console.aliyun.com/#/model-market)查看和体验这些模型。
说明
千问-Turbo、千问-Plus、千问-Max 模型的快照版本（例如qwen-plus-1220）仅兼容“函数调用”及“知识检索增强”工具。模型的兼容性以实际运行结果为准。

| --- | --- |
| 千问-Turbo | qwen-turbo |
| 千问-Plus | qwen-plus |
